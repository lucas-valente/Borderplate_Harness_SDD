# Matriz de Cobertura - Testes por Dominio

## Objetivo

Consolidar cobertura planejada x executada por dominio e por tipo de teste no modulo-alvo.

## Matriz

| Dominio | Arquivos mapeados | Unit | Integracao | E2E | Contrato/Integracao | Seguranca/Isolamento | Status |

| 02-dominio-critico-b | 0 | 0% | 0% | 0% | 0% | 0% | PENDENTE |
| 03-dominio-operacional-c | 0 | 0% | 0% | 0% | 0% | 0% | PENDENTE |

## Gate de cobertura por arquivo alterado

- statements >= 85%
- functions >= 85%
- lines >= 85%
- branches >= 65%

## Observacoes

- Esta matriz deve ser atualizada junto com os planos de dominio.
- Divergencias devem ser registradas no risco tecnico.
